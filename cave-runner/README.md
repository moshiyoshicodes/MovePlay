# Cave Runner

Cave Runner is a 3D endless runner game built with Three.js and Vite. Run, slide, and jump through obstacles while collecting coins. Game progress (high scores, coins, and character unlocks) is saved locally in your browser via `localStorage`.

Originally created by [tope-olajide](https://github.com/tope-olajide/cave-runner), adapted for [MovePlay](https://github.com/moshiyoshicodes/MovePlay).

## Built with

- [Three.js](https://threejs.org/) — 3D rendering
- [Vite](https://vitejs.dev/) — dev server and build tool

## Installation

Requires [Node.js](https://nodejs.org/) (v16+ recommended).

```bash
cd cave-runner
npm install
```

## Running locally

```bash
npm run dev
```

Open the URL Vite prints (typically `http://localhost:3000` or `http://127.0.0.1:5173`). No environment variables or backend setup required.
